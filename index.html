Projects

Cricket MatchMaker

A web application designed to help users co-ordinate between friends for playing cricket or any sports in general. Users register an account, input their availability and schedule can can add friends. If any of their friends are available to play at the same time, they are notified.

LIDAR

Embedded measurement system built using a TI MSP432 uC and ToF sensors to acquire information about the surrounding area. Mapped spatial information is stored and communicated to a personal computer for graphical presentation using the Python Open3D library.

HitchHiker's Guide to SI4

A data structure and algorithm visualizer designed specifically for students to understand algorithms through visualization. The application allows users to input custom arrays and go over single iterations, something that is found regularly in coding questions.
